# Day 24: PCI 枚举 + e1000 网卡

> 对照: reference/linux-7.1/drivers/pci/, e1000

## 1. 目标
- PCI 配置空间访问(0xCF8/0xCFC), 扫描 vendor/device
- e1000 MMIO 映射, 设备复位, TX 描述符环, 发送以太网帧
- **验收**: e1000 send OK

## 2. 关键实现
- `arch/x86/pci.c` — pci_read32(addr|bus|dev|func|offset), pci_scan 枚举 bus0
- `arch/x86/e1000.c` — e1000_init: 找vendor=8086/device=100E→读BAR0→map_mmio→复位→TX环→发送测试包
- TX 描述符(16B): addr(包物理地址), length, cmd(EOP|RS), sta(DD)

## 3. PCI Bus Mastering
- **问题**: send TIMEOUT(DD bit 不置位)
- **根因**: PCI 设备 Bus Mastering 未开启(DMA 无法工作)
- **修复**: `pci_write32(0,dev,0,0x04, cmd|(1<<2))` 设置 COMMAND 寄存器 bit2
